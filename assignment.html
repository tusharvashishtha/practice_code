Definition/ Description list.

Html
HTML stands for HyperText Markup Language. It is a markup language used for creating web pages and other information that can be displayed in a web browser.
CSS
CSS stands for Cascading Style Sheets. It is a style sheet language used to describe the presentation of a document written in HTML or XML, including the layout, colors, fonts, and other visual aspects of the document.
JavaScript
JavaScript is a high-level programming language used to create interactive and dynamic web pages. It is one of the three core technologies used to create modern web applications, alongside HTML and CSS.
Python
Python is a high-level, interpreted programming language that is widely used for a variety of applications.
